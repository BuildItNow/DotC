# DotC
DotC是一款通用的IOS框架，特性包含：<br/>
* 网络模块
* 缓存模块
* 网络API测试数据模块
* 图片缓存模块
* 数据中心模块
* EventEmiter组件
* Delegator组件
* DictionaryWrapper组件
* MJRefreshList 通用刷新Controller组件
* 通用工具组件：DeviceUtil,SystemUtil,AlertUtil,HUDUtil ... <br/>
通过DotC提供的核心功能，使得IOS工程能迅速的搭建和开发<br/>
